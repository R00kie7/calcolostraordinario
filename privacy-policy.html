4. Misure di sicurezza e protezione dati

Adottiamo misure tecniche e organizzative appropriate per proteggere i dati locali da perdita accidentale, uso illecito, accesso non autorizzato, divulgazione, alterazione o distruzione.

Archiviazione locale sicura

Dati sul dispositivo

Tutti i dati sono archiviati localmente sul tuo dispositivo utilizzando le API sicure di Android

Protezione del sistema

I dati sono protetti dal sistema di sicurezza Android (sandbox app, permessi utente)

Backup locali

Sistema di backup integrato per salvare i tuoi dati su file locali, sempre sotto il tuo controllo

Eliminazione completa

Funzione di eliminazione dati per rimuovere completamente tutte le informazioni quando desideri

5. Dati raccolti dall'app

L'app "Calcolo Straordinario" raccoglie e archivia esclusivamente in locale sul tuo dispositivo i seguenti dati:

Dati lavorativi inseriti dall'utente

  • Orari di lavoro: Orario di entrata, uscita, pause pranzo
  • Calcoli straordinari: Ore di straordinario calcolate automaticamente
  • Impostazioni personali: Orario contrattuale, configurazione turni
  • Ferie e permessi: Giorni di ferie utilizzati e disponibili
  • Note personali: Eventuali annotazioni aggiunte alle giornate lavorative

Dati tecnici dell'app

  • Preferenze app: Tema scuro/chiaro, lingua selezionata
  • Configurazioni: Impostazioni personali dell'app (completamente locali)

Principio di minimizzazione

L'app raccoglie SOLO i dati strettamente necessari per fornire le funzionalità di calcolo e gestione presenze. Nessun dato viene trasmesso a server esterni.

Dati NON raccolti: L'app NON raccoglie dati di localizzazione GPS, contatti, fotocamera, microfono, SMS, chiamate o altri dati sensibili del dispositivo. Non vengono raccolti dati identificativi personali (nome, cognome, email, numero telefono).

6. Dati tecnici di sistema Android

Il sistema operativo Android può raccogliere automaticamente alcuni dati tecnici per il funzionamento del dispositivo:

Crash reports e diagnostica (gestiti da Google)

  • Rapporti crash: Se l'app si blocca, Android può inviare automaticamente a Google informazioni tecniche anonime (solo se hai abilitato questa funzione nelle impostazioni Android)
  • Dati tecnici anonimi: Versione Android, modello dispositivo, versione app (senza dati personali)
  • Controllo utente: Puoi disabilitare questa funzione nelle impostazioni Android → Privacy → Diagnostica
Nota importante: Questi dati tecnici sono gestiti direttamente da Google tramite Google Play Services e non sono controllati dall'app. Per maggiori informazioni, consulta la Privacy Policy di Google.

7. Finalità e base giuridica del trattamento

Finalità principali

  • Fornitura del servizio: Calcolo straordinari, gestione presenze e orari di lavoro
  • Backup e ripristino: Permettere il salvataggio e recupero dei dati in caso di cambio dispositivo o reset
  • Personalizzazione: Salvare le tue preferenze di utilizzo dell'app (tema, lingua)

Base giuridica

Il trattamento dei dati è basato su:

  • Consenso (Art. 6.1.a GDPR): Utilizzi volontariamente l'app e inserisci i tuoi dati di lavoro
  • Esecuzione di un contratto (Art. 6.1.b GDPR): Fornitura delle funzionalità dell'app che hai richiesto
  • Legittimo interesse (Art. 6.1.f GDPR): Miglioramento e stabilità dell'app tramite crash reports anonimi

Nessuna profilazione

L'app NON effettua profilazione, decisioni automatizzate o tracciamento comportamentale. I tuoi dati sono utilizzati esclusivamente per le funzionalità di calcolo e gestione presenze.

8. Trasferimento dati e condivisione

Nessun trasferimento dati

I tuoi dati NON vengono MAI trasferiti al di fuori del tuo dispositivo, né in Italia né all'estero.

Nessun server remoto - Non utilizziamo cloud storage o server per archiviare i tuoi dati

Controllo totale - Sei l'unico che ha accesso ai tuoi dati di lavoro

Eccezioni (solo dati tecnici anonimi)

Gli unici dati che potrebbero essere trasmessi sono:

  • Crash reports a Google Play Console: Solo se hai abilitato "Invia dati di diagnostica" nelle impostazioni Android
  • Contenuto: Dati tecnici anonimi (versione app, modello dispositivo, log errori) senza informazioni personali
  • Finalità: Permettere allo sviluppatore di risolvere bug e migliorare la stabilità dell'app
Backup locali: Se crei un backup utilizzando la funzione integrata nell'app, il file viene salvato nella memoria del tuo dispositivo o nella posizione che scegli tu (cartella Download, SD card, ecc.). Rimani sempre tu responsabile della gestione e sicurezza di questi file di backup.
